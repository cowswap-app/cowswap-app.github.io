What is CoW Swap?

CoW Swap is the first interface built on top of CoW Protocol and a reference implementation for its intent-based trading UX. Verify that role in the official CoW Protocol docs before launching the app.

How CoW Swap works

  1. Open the verified appUse swap.cow.fi and check the domain before connecting a wallet.
  2. Connect your walletChoose an injected wallet, Safe, WalletConnect-compatible mobile wallet, or supported hardware wallet path.
  3. Set the orderPick the network, sell token, buy token, amount, and order type such as swap, limit, or TWAP.
  4. Review and signCheck output, expiry, slippage, and approval prompts, then sign the intent from your wallet.

Before you swap

Intent-based execution

Instead of sending an immediate on-chain swap transaction, users sign an intent and solvers compete to execute it through available routes.

Built for user protection

The protocol documentation highlights MEV protection and price improvement as core benefits of the CoW Protocol model.

Familiar DeFi flow

The app supports common wallet paths and order styles while routing through CoW Protocol's solver-based settlement design.

CoW Swap FAQ

Is cow swap the official app?

The action interface is hosted at swap.cow.fi; always verify the domain before connecting a wallet.

How does cow swap execute trades?

Users sign trade intents, and CoW Protocol solvers compete to find execution paths rather than relying only on a direct pool swap.

Does cow swap support limit orders?

Yes. The CoW Protocol docs list market swaps, limit orders, TWAP orders, native-token flows, and cross-chain swap options.

What wallets work with cow swap?

The docs mention injected wallets, Safe, WalletConnect v2 wallets, and hardware-wallet paths through supported wallet integrations.

Is cow swap financial advice?

No. Review token, network, approval, slippage, and order details yourself before signing; DeFi transactions carry execution and smart-contract risk.
